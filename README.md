Projeto de um caixa eletronico em C#.NET 6

Está dividido em 5 pastas com a evolução na implementação, aplicando conceitos de delegate, eventos e expresões lambda em C#

A versão final do projeto contem uma lista com varios clientes que tem agencia e conta. 
O sistema guarda o saldo de cada cliente (em memoria) e permite que o mesmo realize depositos, saques e consulte o extrato.
Interface grafica feita em Windons forms.

Para executar sete mutiplos projetos de inicialização. tanto a API quanto o Front da aula 5
